# Digital_Design flow using Cadence Tools

## Overview
Implemented a complete RTL-to-GDSII digital design flow using Cadence EDA tools on TSMC 65nm technology.

## Flow Covered
- RTL simulation and verification  
- Logic synthesis  
- Floorplanning, placement and routing  
- Clock Tree Synthesis (CTS)  
- Static Timing Analysis (STA)  
- Power analysis  
- Hold violation fixing in Tempus  

## Tools Used
- Cadence Xcelium  
- Cadence Genus  
- Cadence Innovus Stylus  
- Cadence Tempus  
- TCL Scripting  

## Key Learnings
- Hands-on exposure to ASIC implementation flow  
- Timing closure and physical design constraints  
- Process-aware design considerations in 65nm technology
